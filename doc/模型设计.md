# 模型设计

### 用户模型

| 字段 | 类型     | 解释     | 约束       | 处理                          |
| ---- | -------- | -------- | ---------- | ----------------------------- |
|      | char     | 手机号   | 唯一，索引 | 格式：11位纯数字，            |
|      | char     | 邮箱     | 唯一，索引 | 格式   ：登录名@主机名.域名， |
|      | char     | 密码     |            | 加密处理                      |
|      | char     | 真实姓名 |            |                               |
|      | char     | 身份证号 | 唯一       |                               |
|      | char     | 昵称     |            |                               |
|      | char     | 头像     |            |                               |
|      | datetime | 注册时间 |            |                               |
|      | active   | 是否可用 | 默认为1    | 布尔类型                      |
